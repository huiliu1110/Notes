### Command line
Understand some basic commands: 
- cd:     change directory
- ls:     list files
- pwd:    working directory
- rm:     remove file
- mkdir:  make directory
- rmdir:  remove directory
- cat:    show file content
- less:   show file content by page
- man:    command manual
- echo:   repeat input
